This is what I've worked on (and documented) till date

Clicking on the links will take you to github repositories

HTTP Load Balancer

This is something I'm currently working on. The aim is to implement an HTTP, i.e. layer 7 load balancer in C++ using libcurl and libhttpserver. This is part of a SURP project under Prof Manjesh Hanawal. The repository is currently private and I'll link it as soon as it goes public.

OCDE

As part of a course project, a team of four (including me) created this online coding and development environment. We've handcrafted everything, from the UI to the schemas. The code is available in these two repositories (for some reason, we decided seperating frontent and backend into individual repos was a good idea).

IITB-proc

Another group course project took the same team who implemented OCDE to code a 16 bit processor using vhdl. The design is handcrafted and as fast as we could think was possible (:
For ease of testing, we coded a comprehensive testbench supporting I/O and provided a python script to convert human readable assembly code into binary instructions.

SOT

Another course project which me and my Friend Raj Aryan Agrawal worked on together is implementing a paper on a modern compressive transformation technique called sparse orthogonal transforms. The code we wrote and the results we obtained with it are in the repository. The paper can be checked out here.

Mastermind

I, with the help of 2 friends of mine, tried to encode a mastermind game player into SAT so that good practical results could be obtained on the minimum moves problem (which is NP hard). This was part of an assignment which we got overindulged in and tried to do better than required. The encoding we came up with was a sub optimal version of a published one.

Cryptography (SoS)

This was my first introduction to Cryptography as a Summer of Science project. The repositories contains brief notes I made during my 2 month long study of the topic. I've explored the field further but have not taken the time to document my progress. Part of which is because I lost the tex file for the initial project (who knows why I didn't push it then).
